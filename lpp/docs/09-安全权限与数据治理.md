# LPP 安全权限与数据治理

更新时间：2026-05-22

本文档承接 ECC 的审查和安全阶段，记录 token 域、角色权限、客户隐私、敏感字段、审计和风控要求。

## 1. 安全边界

| 边界 | 规则 | 当前风险 |
| --- | --- | --- |
| platform token | 只用于平台登录、租户选择、空间汇总、签发 admin token。 | 需要防止误用于业务写接口。 |
| tenant token | 只用于 `/api/client/v1/*`、`/ws/client`、`/hubs/voicecall`。 | 不能调用 admin API。 |
| admin token | 只用于 `/api/admin/v1/*`。 | 需要逐接口审查。 |
| 客户数据 | 按角色控制可见范围。 | 客户画像和工单字段需服务端确认。 |
| 客服会话 | 管理员和所有者只读查看。 | 只读输入区和发送 service 需双重限制。 |
| 审计数据 | 管理动作必须可追踪。 | 冻结、禁言、分配客服、审批需要审计字段。 |

## 2. 角色权限矩阵

| 能力 | 客户 | 普通成员 | 客服 | 管理员 | 所有者 |
| --- | --- | --- | --- | --- | --- |
| 个人聊天 | 是 | 是 | 是 | 是 | 是 |
| 好友申请 | 是 | 是 | 是 | 是 | 是 |
| 企业通讯录 | 受限 | 是 | 是 | 是 | 是 |
| 客户管理 | 否 | 否 | 受限 | 是 | 是 |
| 客服接待 | 否 | 否 | 是 | 管理/只读 | 管理/只读 |
| 客户画像 | 受限 | 否/受限 | 服务所需字段 | 管理字段 | 管理字段 |
| 群治理 | 否 | 否/受限 | 部分 | 是 | 是 |
| 会话审计 | 否 | 否 | 否/受限 | 是 | 是 |
| 企业配置 | 否 | 否 | 否 | 部分 | 是 |

## 3. 敏感数据治理

| 数据 | 风险 | 处理原则 |
| --- | --- | --- |
| 手机、邮箱 | 个人隐私 | 非本人视角默认脱敏。 |
| token、cookie、密码、验证码 | 安全凭证 | 日志、脚本、报告全部脱敏。 |
| 客户画像 | 客户隐私和经营数据 | 按角色和接待场景展示。 |
| 账户余额、入金、交易 | 高敏商业数据 | 默认只在必要客服/管理场景展示。 |
| 工单和投诉 | 服务敏感信息 | 限制查看范围，关键操作审计。 |
| 风控命中 | 高敏治理数据 | 管理员/所有者查看，客服按需提示。 |
| 审计日志 | 内部治理数据 | 只对管理员/所有者开放。 |

## 4. 高风险动作审计

| 动作 | 是否需要审计 | 必要字段 |
| --- | --- | --- |
| 分配客服 | 是 | 操作者、客户、客服、时间、原因。 |
| 审批加入企业 | 是 | 操作者、申请人、结果、时间、拒绝原因。 |
| 好友申请处理 | 建议 | 操作者、申请、结果、时间。 |
| 群全员禁言 | 是 | 操作者、群、开关、时间、原因。 |
| 群单人禁言 | 是 | 操作者、群、成员、时长、原因、时间。 |
| 冻结/解冻会话 | 是 | 操作者、会话、动作、原因、时间、影响范围。 |
| 接管客服会话 | 是 | 操作者、原客服、新客服、会话、时间。 |

## 5. 审查清单

- 是否存在 tenant token 调 admin API。
- 是否存在客户可见内部工作台。
- 是否存在管理员只读会话仍可发送消息。
- 是否存在客户画像字段无角色控制。
- 是否存在敏感日志未脱敏。
- 是否存在重复点击导致重复审批、重复接管或重复发送。
- 是否存在服务端拒绝后客户端仍显示成功。
- 是否存在脚本输出 token 或修改真实数据而无提示。
