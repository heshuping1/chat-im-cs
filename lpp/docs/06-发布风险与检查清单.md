# LPP 发布风险与检查清单

更新时间：2026-05-23

当前发布判断：不建议直接交付。原因是自动化测试虽已通过，Android 设备 `ce8c42ac` 已识别并安装 release 模式可安装包，且已完成部分单机真机验证，但 Android P0 主链路、多角色、在线客服端到端、推送后台和大数据性能专项仍未完成。

## 1. 发布前最低门槛

发布或对外测试前至少满足：

- S0 问题为 0。
- S1 问题有修复计划或明确延期确认。
- `flutter analyze` 通过或有明确豁免。
- `flutter test ../scripts/mobile/test/flutter` 通过或有明确豁免。
- Android 真机 P0 路径完成验收。
- admin token 和 tenant token 没有混用。
- 管理员/所有者只读查看客户会话时不能发言。
- 客户看不到内部工作台、客户池、内部审计和企业治理入口。
- 音视频、推送、后台能力如果纳入交付，必须有真机专项记录。

## 2. 风险等级

| 等级 | 定义 | 示例 | 处理 |
| --- | --- | --- | --- |
| S0 | 阻塞交付或存在严重安全风险。 | 登录失败、消息全失败、token 混用、客户看到内部数据。 | 必须立即修复。 |
| S1 | 关键流程失败但有临时替代路径，或真实可用性未知。 | 好友闭环异常、客服无法接管、群禁言权限错误、通话未验收。 | 本轮修复或明确延期。 |
| S2 | 体验问题或边界异常。 | loading 缺失、重复点击、空态不完整、文案不清。 | 排期修复。 |
| S3 | 低频、文案、视觉细节。 | 间距轻微不齐、非关键提示不统一。 | 可延期。 |

## 4. 发布前检查项

### 4.1 账号和空间

- 邮箱登录成功。
- 微界号登录成功。
- 快速登录在测试环境可用，生产环境不暴露。
- 空间选择成功。
- 切换空间成功。
- 退出登录成功。
- token 刷新不导致异常退出。

### 4.2 IM 主链路

- 会话列表正常加载。
- 文本消息发送成功。
- 图片、语音、文件至少一种媒体消息可用。
- 发送失败可重试。
- 搜索可以跳转原消息。
- 收藏可以回到原消息或给出明确失效提示。
- 翻译只翻译对方消息。
- Gateway 断线重连后不重复、不丢失关键消息。

### 4.3 好友和客户

- 搜索加好友正常。
- 扫码加好友正常。
- 申请通过后双方状态正确。
- 客户管理列表正常。
- 标签筛选正常。
- 分配客服正常。
- 客户画像缺字段时不显示假数据。

### 4.4 群管理

- 创建群正常。
- 群成员列表正常。
- 客户作为普通成员权限正确。
- 全员禁言权限正确。
- 单人禁言时长和原因正确。
- 群申请审批状态正确。
- 服务端拒绝时客户端不显示成功。

### 4.5 在线客服和工作台

- 客服状态切换正常。
- 待接入列表正常。
- 认领、接管、关闭正常。
- 管理员和所有者查看会话为只读。
- 在线客服概览数据正常或缺失时显示占位。
- 风控和审计入口权限正确。

### 4.6 通话和推送

- 语音通话成功。
- 视频通话成功。
- 来电提醒正常。
- 后台和锁屏行为可接受。
- 通知点击能进入正确空间和会话。

### 4.7 性能和稳定性

- 冷启动时间有记录。
- 会话列表长列表滚动有记录。
- 聊天页长消息滚动有记录。
- 弱网发送和重连有记录。
- 内存占用有记录。

## 5. 不能发布的情况

- 任一主要角色无法登录。
- 普通消息无法发送。
- 客户能进入管理员或所有者工作台。
- admin API 使用普通 tenant token。
- 管理员/所有者能以第三人视角向客户会话发消息。
- 群禁言权限与产品规则相反。
- 客服无法处理客户咨询。
- App 在 P0 路径频繁崩溃、白屏或卡死。
- 纳入交付范围的通话/推送能力没有真机验收。

## 6. 可延期项

以下内容可延期，但必须在功能矩阵和版本交付报告中标记：

- 深色模式完整打磨。
- 图片/视频压缩优化。
- 消息撤回后重新编辑。
- 面对面建群。
- 营销工具栏。
- PC 端完整在线客服工作台。
- 非 P0 性能优化。
