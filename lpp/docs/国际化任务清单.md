## 每个模块 PR 的最小验收标准

```text
1. 默认 zh-CN 下视觉和功能不变
2. 新增文案 key 有 6 语言值
3. 缺 key 有 fallback
4. 不翻译用户内容、服务端业务内容、协议值
5. Domain/Data/Runtime 不依赖 i18n Provider、React Hook、t()
6. 领域状态和错误码先映射为 key，再由 Presentation 调用 t()
7. 翻译后文案不写入 store、cache、API request、IPC payload
8. typecheck、lint、build 至少按影响范围通过
```
