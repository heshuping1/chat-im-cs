# LPP 接口与服务端依赖

更新时间：2026-05-22

最后核对方式：静态扫描 Flutter 代码中的 Dio endpoint、WebSocket 路径和通话 hub 路径。本文档记录客户端已调用或依赖的接口，不代表接口已全部联调通过。

如果要按“功能异常”查接口，优先使用 [功能接口映射表](14-功能接口映射表.md)。本文档偏接口域和服务端依赖总览，映射表偏排查入口。

## 1. 接口域和 token

| 接口域 | 用途 | Token |
| --- | --- | --- |
| `/api/platform/v1/*` | 平台登录、租户、空间、平台账号。 | platformToken |
| `/api/client/v1/*` | IM、通讯录、客户、群、客服、设置。 | tenant accessToken |
| `/api/admin/v1/*` | 管理员/所有者工作台、客户池、治理、审计。 | admin accessToken |
| `/ws/client` | 客户端实时消息和空间事件。 | tenant accessToken |
| `/hubs/voicecall` | 音视频通话信令。 | tenant accessToken |

## 2. token 规则

- platformToken 不用于普通业务写操作。
- tenant accessToken 不调用 `/api/admin/v1/*`。
- admin accessToken 只用于 admin API。
- admin token 由 platform token 通过目标 tenantId 签发。
- 日志、报告、脚本输出必须脱敏 token。

## 6.1 客户端普通群发复用接口

客户端消息首页 `+` 菜单中的普通“群发”不需要服务端新增批量任务接口。客户端逐个目标复用现有普通 IM 接口：

| 目标 | 复用接口 | 说明 |
| --- | --- | --- |
| 好友 | `POST /api/client/v1/direct-chats` | 创建或复用与好友的直聊会话。 |
| 好友 | `POST /api/client/v1/direct-chats/{chatId}/messages` | 按当前账号身份发送普通单聊消息。 |
| 群聊 | `POST /api/client/v1/groups/{groupId}/messages` | 按当前账号身份发送普通群消息；群禁言、非成员、无权限等由接口返回。 |
