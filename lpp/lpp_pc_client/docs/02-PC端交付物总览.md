# PC 端交付物总览

状态：当前有效入口

日期：2026-06-06

适用范围：`lpp_pc_client`

## 1. 结论

PC 端当前可作为普通 IM 基础闭环和业务模块入口继续交付。本文档只保留交付范围索引、发布风险和下一步阅读路径。

详细功能状态见 `03-PC端功能矩阵.md`，发布执行见 `06-PC端发布文档.md`。

## 4. 发布前最低验证

```bash
npm run typecheck
npm run test:core
```

普通 IM 发布候选专项：

```bash
npx playwright test tests/browser/workspace-smoke.spec.ts -g "PC ordinary IM closure|PC ordinary IM stage two|PC ordinary IM stage three|PC ordinary IM stage four|updates IM unread|marks incoming messages" --workers=1
```

本机 PowerShell 禁止 `npm.ps1` 时使用：

```bash
cmd /c npm.cmd run typecheck
cmd /c npm.cmd run test:core
```

## 5. 暂停发布条件

| 风险 | 判断 |
| --- | --- |
| 消息不可用 | 收不到消息、消息空白、实时消息不展示 |
| 未读错误 | 自发消息产生未读、未读错误增加或错误清除 |
| 媒体错误 | 图片/文件/视频发送失败无提示或附件丢失 |
| 操作回退 | 删除、撤回、收藏、转发后 refetch 状态回退 |
| 群展示异常 | 群聊 sender 大面积显示 UUID |
| 错误不可定位 | 401/403/500 无 requestId 或诊断记录 |
| 合同失效 | 服务端返回结构变化导致 normalizer 大面积失效 |
| 假实现风险 | 服务端缺口被前端假逻辑掩盖 |
