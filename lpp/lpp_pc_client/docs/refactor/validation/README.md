---

## 记录原则

1. 核心链路变更必须留下验证证据。
2. 不能运行完整测试时，必须记录原因和替代验证。
3. 诊断日志要说明 Codex 如何检索和复盘。
4. 验证记录只写事实，不写泛泛总结。
