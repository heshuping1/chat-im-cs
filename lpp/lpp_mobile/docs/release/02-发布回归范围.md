# APP 发布回归范围

状态：当前有效

日期：2026-06-06

适用范围：`lpp_mobile`

## 1. P0 回归

| 路径 | 必测内容 |
| --- | --- |
| 登录和空间 | 邮箱/LPP 号登录、租户选择、个人空间、企业空间、退出登录、token 刷新。 |
| 消息 | 会话列表、文本、图片、文件、语音、失败重试、未读、已读、搜索、收藏、翻译。 |
| 好友 | 搜索、扫码、申请、同意、拒绝、通过后会话出现。 |
| 通讯录 | 组织、成员、客户、群、角色可见范围。 |
| 群 | 创建群、添加成员、群资料、禁言、申请、公告、退出/解散。 |
| 在线客服 | 排队、接入、人工接管、继续沟通、关闭、历史只读、来源渠道、客户摘要。 |
| 管理 | 管理员/所有者只读查看、客户管理、客服中心、群监管入口。 |
| 设置诊断 | 通知、隐私、安全、语言、诊断导出、退出登录。 |

## 2. 设备专项

| 专项 | 必测内容 |
| --- | --- |
| Android 真机 | 安装、启动、登录、前后台切换、权限弹窗、截图证据。 |
| 弱网 | 断网提示、恢复、Gateway 重连、离线补偿、发送重试。 |
| 推送 | 前台、后台、锁屏、杀进程、点击进入正确空间和页面。 |
| 通话 | 呼叫、接听、拒绝、挂断、后台、锁屏、音频路由。 |
| 媒体 | 相机、相册、文件、录音、上传失败、鉴权失效。 |
| 性能 | 500+ 会话、长聊天、低端机、长时间后台恢复。 |
