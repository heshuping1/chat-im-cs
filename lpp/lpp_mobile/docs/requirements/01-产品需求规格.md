# App 产品需求规格

更新时间：2026-06-06

完整跨端需求见 `../../../docs/01-需求规格说明书.md`。本文件只保留移动端 App 当前执行口径。

## 1. 产品定位

LPP App 是企业 IM + 在线客服 + 客户经营的移动端应用。它优先服务 Android/iOS 移动场景：快速登录、即时沟通、轻量接待、扫码、通话、移动通知、基础客户查看和低频管理操作。

App 不承载 PC 端的完整客服三栏工作台、完整客户交易表、资金流水明细、质检审计大屏、批量运营分析和桌面能力。

## 2. 空间和角色

| 对象 | App 口径 |
| --- | --- |
| 个人空间 | 可登录、展示个人会话、好友、群和个人设置。 |
| 企业员工空间 | 支持企业成员、组织、群、客户、客服工作台和管理入口。 |
| 企业客户空间 | 可沟通、可发起客服咨询，不暴露内部管理入口。 |
| 在线客服临时会话 | 与聊天承载复用，但入口、状态和权限独立于普通 IM。 |

| 角色 | App 能力边界 |
| --- | --- |
| 客户 | 聊天、群、客服咨询、个人资料；不能看到内部工作台、客户池、审计和治理。 |
| 普通成员 | 基础聊天、组织通讯录、权限内客户沟通。 |
| 客服 | 客服工作台、在线/忙闲状态、接待会话、客户摘要、快捷回复。 |
| 管理员 | 客户管理、客服中心、群监管、企业管理等轻量管理入口。 |
| 所有者 | 全局治理和配置入口，拥有更高管理权限。 |

## 3. App/PC 边界

| 功能域 | App 承载 | PC/后续承载 |
| --- | --- | --- |
| 普通 IM | 会话列表、聊天、媒体、搜索、收藏、客户端普通群发。 | 更高密度会话列表、资料侧栏、效率工具。 |
| 在线客服 | 移动接待、接入、人工接管、关闭、客户摘要。 | 多栏队列、客户全景、SLA、团队协同。 |
| 客户画像 | 基础资料、标签、账户概况、临时订单、工单摘要。 | 交易、资金、触达、KYC、合规、设备、完整工单。 |
| 客户管理 | 核心查询、标签、分配客服、轻量管理。 | 完整 CRM、批量操作、运营分析。 |
| 群治理 | 核心群设置、禁言、申请、公告、应急治理。 | 更完整治理、审计和批量操作。 |
| 音视频 | 移动端语音/视频、来电、权限、后台。 | PC 通话需独立方案。 |
| 推送 | 移动推送、前后台通知、点击跳转。 | 桌面通知和点击路由。 |

## 5. 当前版本范围

### 必须优先闭环

- 登录、空间、token 域隔离。
- 普通 IM 会话、聊天、媒体、本地缓存、未读和 Gateway 恢复。
- 好友、通讯录、群聊与基础群治理。
- 在线客服移动接待、接入、人工接管、关闭和历史只读。
- 管理员/所有者只读查看客户会话。
- Android 真机 P0 路径、弱网、权限、通话和推送专项。

### 后续范围

- 完整 CRM、交易/资金全量明细、质检审计、运营分析。
- 深色模式完整打磨、复杂营销工具栏、面对面建群。
- PC 专属桌面能力和完整管理扩展。
