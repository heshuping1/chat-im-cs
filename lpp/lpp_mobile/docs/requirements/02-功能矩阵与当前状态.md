## 1. 状态口径

| 状态 | 含义 |
| --- | --- |
| 可用 | 有入口、实现和自动化/接口/真机证据，核心场景可交付。 |
| 代码已验证 | 代码、模型、Provider、接口参数或测试能证明核心规则，但仍缺真实账号、真机或服务端数据。 |
| 代码已接入/待联调 | 前端已调用正式接口，但字段、权限、错误码或真实数据尚未完整联调。 |
| 部分可用 | 只覆盖局部流程或局部状态。 |
| 待联调/人工 | 只能靠真实账号、多端、设备能力、服务端数据或人工体验确认。 |
| 需服务端 | 缺接口、字段、事件、权限或服务端语义。 |
| 未实现 | 无可用入口或只有占位。 |
