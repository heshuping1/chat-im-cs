# App 接口与服务端依赖

更新时间：2026-06-06

完整接口总账见 `../../../docs/05-接口与服务端依赖.md`，功能到接口排查见 `../../../docs/14-功能接口映射表.md`，字段合同见 `../../../docs/api-contracts`。

## 1. 接口域和 token

| 接口域 | 用途 | Token |
| --- | --- | --- |
| `/api/platform/v1/*` | 登录、租户、空间、平台账号。 | platformToken |
| `/api/client/v1/*` | IM、通讯录、客户、群、客服、设置。 | tenant accessToken |
| `/api/admin/v1/*` | 管理员/所有者工作台、客户池、治理、审计。 | admin accessToken |
| `/ws/client` | 客户端实时消息和空间事件。 | tenant accessToken |
| `/hubs/voicecall` | 音视频通话信令。 | tenant accessToken |

## 2. 规则

- platformToken 不用于普通业务写操作。
- tenant accessToken 不调用 admin API。
- admin accessToken 只用于 `/api/admin/v1/*`。
- admin token 由 platform token 通过目标 tenantId 签发。
- 日志、报告、脚本输出必须脱敏 token、cookie、password、验证码。

| 模块 | 关键接口 |

| 通话推送 | `/voicecall/sessions`、`/hubs/voicecall`、`/notifications/devices`、`notification-settings` |

## 4. 服务端待明确

| 能力 | 待明确内容 |
| --- | --- |
| 长期登录态 | 设备级凭证、刷新策略、踢出和重登事件。 |
| 空间未读 | 是否只统计普通 IM，是否混入客服/公告/系统事件。 |
| 客服线程 | 状态字段、终态事件、SLA、来源渠道、客户正在输入。 |
| 客户画像 | KYC、风险、余额、入金、语言、来源、标签和脱敏规则。 |
| 工单 | 当前工单、历史工单、处理状态、负责人、更新时间。 |
| 只读会话 | 管理员/所有者第三人读取权限和写操作拒绝语义。 |
| 群治理 | 企业群和客户自建群区分、禁言同步、审计字段。 |
| 推送 | 前台、后台、杀进程、来电通知、点击跳转闭环。 |
| 通话 | 后台、锁屏、弱网、小窗和音频路由协同。 |
