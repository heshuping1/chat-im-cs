# APP 端需求规格说明

状态：当前有效

日期：2026-06-06

适用范围：`lpp_mobile`

## 1. 文档定位

本文件是 APP 端需求短入口，用于快速判断产品范围、角色边界、当前交付重点和需求是否允许继续推进。详细需求、状态和未实现项分别进入 `requirements/`。

APP 端不是 PC 文档的镜像。它只承载移动 IM、在线客服轻量接待、移动通知、扫码、通话、设置和轻量管理，不把 PC 三栏客服指挥台、完整 CRM、审计大屏、批量运营分析和桌面能力下放到手机端。

## 2. 快速读取

| 需要回答 | 读取文档 |
| --- | --- |
| APP 当前产品边界是什么 | `requirements/01-产品需求规格.md` |
| 新需求是否已有记录 | `requirements/01-APP端需求池.md` |
| 需求状态是否允许开发 | `requirements/03-APP端需求状态流转规则.md`、`constraints/03-需求与设计准入规则.md` |
| 当前实现了什么 | `03-APP端功能矩阵.md` |

| 服务端是否支持 | `05-服务端支持.md` |

## 3. 当前需求口径

| 范围 | APP 端口径 |
| --- | --- |
| 登录与空间 | 邮箱、手机号、星络号、验证码、租户选择、个人空间、企业空间和长期登录态。 |
| 普通 IM | 会话列表、聊天、媒体、搜索、收藏、未读、已读、Gateway 恢复和本地缓存。 |
| 通讯录 | 好友、群、企业成员、组织、客户入口按角色展示。 |
| 群功能 | 创建群、群资料、成员、公告、二维码、禁言、申请、管理员和退出/解散。 |
| 在线客服 | 移动接待、排队、接入、人工接管、关闭、客户摘要和历史只读。 |
| 管理入口 | 管理员/所有者轻量查看客户、客服中心、群监管和企业管理入口。 |
| 设置诊断 | 通知、隐私、安全、语言、聊天设置、诊断导出和反馈。 |
| 通话推送 | Android 优先验证语音/视频、前后台通知和点击跳转。 |

## 4. 开发准入

新增需求必须同时满足：

1. 已登记到 `requirements/01-APP端需求池.md` 或对应需求规格。
2. `需求状态 = 完成`。
3. `设计状态 = 完成`。
4. 已确认服务端 API、字段、事件、权限和错误码支持，或明确标记为 `待服务端`。
5. 已同步 `03-APP端功能矩阵.md`。

不满足准入时，只允许做需求澄清、技术设计、接口确认、风险标记或明确的 P0/P1 缺陷修复。
