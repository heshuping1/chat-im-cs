# APP 端功能矩阵

状态：当前有效

日期：2026-06-06

适用范围：`lpp_mobile`

## 1. 文档定位

本文件是 APP 端功能状态短入口。详细矩阵继续维护在 `requirements/02-功能矩阵与当前状态.md`；未实现项、专项验证和服务端阻塞进入 `requirements/02-APP端未实现需求清单.md`。

## 2. 状态口径

| 状态 | 含义 |
| --- | --- |
| 可用 | 有入口、实现和自动化/接口/真机证据，核心场景可交付。 |
| 代码已验证 | 代码、模型、Provider、接口参数或测试能证明核心规则，仍缺真实账号、真机或服务端数据。 |
| 代码已接入/待联调 | 前端已调用正式接口，但字段、权限、错误码或真实数据尚未完整联调。 |
| 部分可用 | 只覆盖局部流程或局部状态。 |
| 待真机 | 主要差距是真机、多账号、多端或设备能力验证。 |
| 待服务端 | 缺接口、字段、事件、权限或服务端语义。 |
| 未实现 | 无可用入口或只有占位。 |
