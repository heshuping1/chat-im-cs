# APP 端服务端支持

状态：当前有效

日期：2026-06-06

适用范围：`lpp_mobile`

## 1. 文档定位

本文件是 APP 端服务端支持短入口。字段合同以 `../../docs/api-contracts` 为准；详细接口分组可继续参考 `solutions/04-接口与服务端依赖.md`。

## 2. API 支持确认规则

新增或变更需求进入开发前，必须确认：

1. endpoint、method、token 域和权限角色。
2. 请求字段、响应字段、枚举和空值语义。
3. 错误码、错误文案 key、requestId 和重试策略。
4. Gateway/推送/通话事件是否需要同步。
5. 是否需要 admin token、tenant token 或 platform token。
6. 是否有服务端未完成项；未完成时需求状态不能写成可交付。

## 3. Token 域

| 接口域 | 用途 | Token |
| --- | --- | --- |
| `/api/platform/v1/*` | 登录、租户、空间、平台账号。 | platformToken |
| `/api/client/v1/*` | IM、通讯录、客户、群、客服、设置。 | tenant accessToken |
| `/api/admin/v1/*` | 管理员/所有者工作台、客户池、治理、审计。 | admin accessToken |
| `/ws/client` | 客户端实时消息和空间事件。 | tenant accessToken |
| `/hubs/voicecall` | 音视频通话信令。 | tenant accessToken |

## 4. 当前重点待支持项

| 编号 | 能力 | 服务端需确认 |
| --- | --- | --- |
| AUTH-SERVER-001 | 登录与空间 | 验证码通道、长期登录态、冻结/停用/无权限错误码。 |
| CS-SERVER-001 | 在线客服 | 工作台 threads/dashboard 对真实客服账号稳定可用。 |
| CS-EVENT-001 | 在线客服事件 | 排队、接入、接管、关闭、未读、SLA 风险、客户正在输入事件。 |
| ADMIN-CS-001 | 管理客服中心 | 客服负载、待回复、未读、超时、平均等待等统计。 |
| ADMIN-CS-002 | 客服绩效跨渠道统计 | `GET /api/admin/v1/customer-service/temp-sessions/stats` 返回 `staffPerformance` 顶层跨 `widget` + `im_direct` 合并 KPI，并通过 `byChannel[]` 下钻渠道；统计有约 5 分钟聚合延迟，APP 只读展示，不作为实时排队/接待状态。 |
| ADMIN-CS-003 | 访客获客来源 | 临时会话列表项、访客列表项和访客详情支持 `acquisition`，APP 展示 `sourcePlatform`、`chatTool`、`deviceType`、`os`、`utm*`、`appVersion`、地区和时区等结构化来源字段；字段全可空，不展示假数据。 |
| PUSH-SERVER-001 | 推送 | 移动推送 payload、deep link、空间和会话跳转字段。 |
| GROUP-SERVER-002 | 群治理 | 群主、管理员、普通成员和客户成员权限拒绝码。 |
| GROUP-SERVER-003 | 群 @ 提醒 | `POST /api/client/v1/groups/{groupId}/messages`、历史消息和 Gateway `msg.new` 需兼容 `mentions[]`；`@所有人` 使用 `userId=00000000-0000-0000-0000-000000000000` 哨兵项，服务端按角色与 `allowMemberAtAll` 校验，普通成员无权限时返回 `403 MSG_AT_ALL_NOT_ALLOWED`。 |
| GROUP-SERVER-004 | 群二维码进群 | 生成/列表/撤销使用 `/api/client/v1/groups/{groupId}/invite-qr`，扫码预览/加入使用 `/api/client/v1/groups/join-by-qr/{token}/preview` 与 `/accept`；服务端只返回 `token` + `qrPayload`，APP 自行渲染二维码。 |
| PRESENCE-SERVER-001 | 在线状态 | 在线、离线、后台、多端聚合和最后活跃时间。 |

完整追踪见 `requirements/02-APP端未实现需求清单.md`。
